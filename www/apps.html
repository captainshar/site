Apps! Apps! Apps!

Make sure you have Master node a Worker or two before installing Apps. Head back to Install for how. Else hit up the Apps page on your Master.

Preview

Before an App can be installed, it has to be previewed. In the future, I'll have a bunch of Apps ready to be installed. For now, we'll have to preview manually.

In the future, I'll have a bunch of Apps period. For now there's just one: Redis.

  1. Select GitHub.
  2. Enter 'klot-io/redis' in owner/project field.
  3. Click the Preview button.
  4. In 10-20 second, Redis will appear in the Apps listing.
  5. Click to view the App so we can Install it.

Labels

Some Apps require that you label nodes to install service on speciallly enabled nodes os just consistently. The App's page allows you to set these labels easily with checkboxes.

  1. In the Redis App.
  2. Click a checkbox under labels to install it on a specfic Node.
  3. It will be automatically saved.
  4. You can uncheck a checkbox to remvoe a label.

For you Kubernetes savy folks, yes this is labeling Nodes through traditional Kubernetes.

Install

When Ready, click Install. You can head over to the Pods page to watch the Pods come up.

Great Success!

Congrats! You now have Redis running in your home cluster.

Currently, you can only access this Redis instance from Kubernetes directly. Check out Development to setup access to your cluster through kubectl.
