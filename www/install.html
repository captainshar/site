Requirements

  • At minimum you'll need two Raspberry Pi's, probably the 3B+.
    • Maybe it'll work with others? I just haven't tried.
    • If you want to do more stuff, use like 3 or 4.
  • You'll also need to be able to wire a Raspberry Pi to your network, ethernet style.
    • This is just for setup, and you only have to do it one at a time.

Process

  1. Download the pi-0.1.img and burn it into an SD card.
    1. I'm fond of Samsung's EVO Select 32Gb cards but use whatever you like.
    2. I'm not an expert in which cards perform best or whatever.
  2. Power Up
    1. Insert SD card into Pi. Wire to your network (ethernet cable). Power up.
    2. Wait a minute or so as the OS expands the filesystem.
    3. Get impatient and go to http://klot-io.local/ anyway, understanding it might not come up right away.
    4. The site will recognize it has default settings and automatically log you in with the password 'kloudofthings'.
    5. Oh btw, you're logging into your new Pi right now.
  3. Configure Master
    1. Set a new password. This'll be both for the GUI/API you're using and the pi account on the Pi (make sure it's like 8 chars long).
    2. Decide whether to enable SSH or leave it disabled. It's up to you, and you can change later through this page.
    3. Enter the WiFi settings for your network or leave it as is to stay wired.
    4. Set the role to Master for this first one and give your cluster a name, lowercase, no spaces (I should have something check that and password too).
    5. Click Config. You should now have to log in with your new password.
    6. Note the site URL changed to (cluster)-klot-io.local. This is the hostname of the Master now and will be your home base going forward.
  4. Join Workers
    1. Once the status is Master, burn another SD card, and plug it into a new Pi, wire to the network, and power up.
    2. Head over the Nodes page. After a few minutes, a Node called "klot-io" willl appear. That's your new Node.
    3. To have it join as a Worker with the same settings as the Master save the name, give it a name and click Join.
    4. To have it join as a Worker with different settings, click it's name and Config on its own site.
    5. Back at the Master's Nodes, page, wait 20 seconds or so, and it'll appear in the Node listing as (name)-(cluster)-klot-io.local
    6. Repeat for as many Workers as needed. Just make sure each name is unique (ya, should probably write something to check this too).

Congrats! You have a Kubrernetes Cluster on Raspberry Pi's! Hit up Apps for what's next.
